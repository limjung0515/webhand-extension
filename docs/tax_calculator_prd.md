# PRD: 라이프이즈세일즈 AI 세무 급여 계산기 (2025)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 라이프이즈세일즈 AI 세무 급여 계산기 |
| **목적** | 사장님과 세무 직원이 실수령액이나 근무시간을 기반으로 복잡한 급여 산출 및 세무 신고 데이터를 즉시 확인하기 위함 |
| **핵심 가치** | 2025년 최신 법령 반영, 실수령액 기반 역산 기능, 명확한 세무 신고 가이드 제공 |

## 3. 핵심 기능 요구사항

### 3.1. 확장 가능한 내비게이션 (Scalable Architecture)
- **다중 계산기 지원**: 급여 계산기 외에도 향후 다양한 세무/노무 계산기가 추가될 수 있는 탭(Tab) 또는 사이드바 내비게이션 구조 채택.
- **페이지 전환 애니메이션**: 계산기 간 전환 시 부드러운 트랜지션 제공.

### 3.2. 급여 계산 모드 (Dual Mode)

#### Mode A: 실수령액 기반 역산 (Reverse Calculation)
- **입력**: 목표 실수령액 (Net Pay)
- **로직**: 
  - 목표 실수령액을 맞추기 위한 [기본급 + 주휴수당 + 비과세 항목]의 최적 조합 산출
  - 해당 급여를 위해 신고해야 할 주당 근로시간 역산
- **출력**: 세전 급여, 공제 총액, 필요한 주당 총 시간

#### Mode B: 근무 시간 기반 계산 (Forward Calculation)
- **입력**: 주당 근무 시간 또는 월 총 근무 시간
- **로직**: 
  - 최저임금(10,030원) 기준 기본급 산출
  - 주휴수당 및 4대 보험 자동 계산
- **출력**: 실수령액(Net Pay), 세전/세후 상세 내역

### 3.2. 세무/노무 로직 (2025 핵심 정책)

1. **최저임금**: 시급 **10,030원** 적용
2. **주휴수당 자동 적용**:
   - 주 15시간 이상 근무 시 반드시 포함
   - 식: `(주당 근무시간 / 40시간) * 8시간 * 시급`
3. **비과세 항목 최적화**:
   - 식대: 최대 20만 원 반영
   - 자가운전보조금: 최대 20만 원 반영
   - 보험료 및 소득세 계산 시 과세 대상액에서 차감하여 절세 효과 반영
4. **4대 보험 자동 분기**:
   - **주 15시간 미만 (월 60시간 미만)**: 고용보험(0.9%)만 징수
   - **주 15시간 이상**: 국민연금(4.5%), 건강보험(3.545%), 장기요양보험(건강보험의 12.95%), 고용보험(0.9%) 징수
5. **소득세/지방소득세**: 간이세액표 기준 적용 (또는 급여의 약 1% 수준으로 간략화)

### 3.3. 결과 출력 및 리포트

- **정산 결과 요약**: 가장 중요한 데이터(실수령액 vs 세전급여)를 상단에 강조
- **상세 테이블**: 
  - **지급 내역**: 기본급, 주휴수당, 비과세 합계, 총 급여액
  - **공제 내역**: 국민연금, 건강보험, 요양보험, 고용보험, 소득세, 지방소득세 총계
- **세무 신고 가이드**: 
  - 신고할 주당 근로시간
  - 월 소정근로시간
  - 적용된 보험 요율 정보 요약

## 4. 데이터 구조 (Types)

```typescript
export interface SalaryInput {
  type: 'NET_TO_GROSS' | 'HOURS_TO_NET';
  value: number; // 실수령액 또는 근무시간
  nonTaxableMeal?: number; // 식대 (기본 20만)
  nonTaxableDriving?: number; // 자가운전 (기본 0)
}

export interface CalculationResult {
  grossPay: number;      // 세전 급여
  basePay: number;       // 기본급
  weeklyRestPay: number; // 주휴수당
  nonTaxableTotal: number;
  deductions: {
    pension: number;     // 국민연금
    health: number;      // 건강보험
    care: number;        // 장기요양
    employment: number;  // 고용보험
    incomeTax: number;   // 소득세
    localTax: number;    // 지방소득세
    total: number;
  };
  netPay: number;        // 실수령액
  reportingHours: number; // 신고할 주당 근로시간
}
```

## 5. 디자인 시스템 (Premium SaaS Look)

- **Base Colors**: `bg-slate-50`, `text-slate-900`
- **Primary Color**: `blue-600` (신뢰감)
- **Accent Color**: `indigo-600`
- **UI 스타일**: 
  - `rounded-3xl`의 부드러운 카드 디자인
  - `shadow-sm` 및 `border-slate-200`으로 깔끔한 경계
  - 데이터 대조를 위한 선명한 테이블 레이아웃
- **인터랙션**: 입력값 변경 시 실시간(Reactive) 계산 결과 업데이트
- **반응형 디자인 (Mobile-First)**: 
  - 모바일에서도 모든 계산기와 결과를 한눈에 보기 편하도록 최적화된 레이아웃.
  - 데스크탑에서는 좌/우 (입력/결과) 배치, 모바일에서는 상/하 스택 배치.
  - 터치 친화적인 입력 필드 및 버튼 크기 확보.

## 6. 개발 요청용 프롬프트 (Master Prompt)

```text
React 19와 TypeScript, Tailwind CSS를 사용하여 '라이프이즈세일즈 AI 세무 급여 계산기'를 만들어줘. 

### 1. 디자인 및 UX (Premium SaaS Look & Responsive)
- **테마**: `bg-slate-50` 배경에 `rounded-3xl` 카드를 사용한 프리미엄 SaaS 스타일.
- **반응형**: **모바일 최우선(Mobile-first)** 레이아웃. 데스크탑은 2컬럼, 모바일은 1컬럼 스택 구조.
- **확장성**: 상단에 탭 기능을 구현하여 향후 다른 세무 계산기를 쉽게 추가할 수 있는 구조로 개발.
- **컬러**: `blue-600`과 `indigo-600`을 메인 포인트로 사용.

### 2. 핵심 계산 로직 (2025년 기준)
- **최저임금**: 10,030원
- **주휴수당**: 주 15시간 이상 시 `(시간/40)*8*시급` 자동 계산.
- **비과세**: 식대 20만, 차량 20만까지 과세대상에서 제외.
- **보험료**: 15시간 이상/미만 여부에 따라 요율 자동 분기 (국민 4.5%, 건강 3.545%, 고용 0.9% 등).
- **역산 기능**: 사용자가 '목표 실수령액'을 입력하면 세전 급여와 근로시간을 역산하여 도출하는 로직 구현.

### 3. 출력 요구사항
- **정산 결과 요약**: 세전/세후 금액 시각적 강조.
- **상세 내역 테이블**: 지급/공제 내역 항목별 표시 (모바일에서는 가로 스크롤 또는 리스트형태로 최적화).
- **세무 신고 가이드**: 신고용 주당 근로시간 및 적용 요율 텍스트 제공.

모든 코드는 한글 기반 UI로 작성하고, 즉시 실행 가능한 형태로 구현해줘.
```
